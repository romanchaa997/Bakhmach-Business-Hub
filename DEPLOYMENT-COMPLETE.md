## üîß QUICK START DEPLOYMENT

### 1. Clone Repository
```bash
git clone https://github.com/romanchaa997/Bakhmach-Business-Hub.git
cd Bakhmach-Business-Hub
```

### 2. Install Dependencies
```bash
pip install -r backend/requirements.txt
pip install -r ml/requirements.txt
```

### 3. Setup Database
```bash
psql -U postgres -d bakhmach_prod -f database/audityzer_schema.sql
psql -U postgres -d bakhmach_prod -f database/debtdefense_schema.sql
psql -U postgres -d bakhmach_prod -f database/bsec_schema.sql
psql -U postgres -d bakhmach_prod -f database/cannabis_schema.sql
```

### 4. Deploy Microservices (Docker)
```bash
docker-compose up -d audityzer
docker-compose up -d debtdefense
docker-compose up -d bsec
docker-compose up -d cannabis
```

### 5. Run Integration Tests
```bash
pytest backend/tests/test_audityzer_integration.py -v
pytest backend/tests/test_debtdefense_integration.py -v
pytest backend/tests/test_bsec_integration.py -v
pytest backend/tests/test_cannabis_integration.py -v
```

### 6. Deploy to Kubernetes
```bash
kubectl apply -f k8s/audityzer-deployment.yaml
kubectl apply -f k8s/debtdefense-deployment.yaml
kubectl apply -f k8s/bsec-deployment.yaml
kubectl apply -f k8s/cannabis-deployment.yaml
```
